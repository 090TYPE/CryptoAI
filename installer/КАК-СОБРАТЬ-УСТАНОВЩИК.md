# CryptoAI Terminal — установщик и подпись

Эта папка делает из приложения один файл-установщик `CryptoAITerminal-Setup-X.Y.Z.exe`
вместо россыпи DLL. Установщик ставит приложение, создаёт ярлыки и убирает часть
проблем с антивирусом. **Полностью** проблему «обезвредили / не запускается» снимает
только покупной сертификат подписи (см. ниже).

## Что нужно один раз
- **Inno Setup** (бесплатный): `winget install JRSoftware.InnoSetup`
- **.NET SDK 8** (уже есть, раз собираешь приложение).
- signtool (для подписи) уже идёт с Windows SDK — у тебя есть.

## Собрать установщик (без подписи)
В PowerShell из корня репозитория:

```powershell
pwsh installer\build-installer.ps1 -Version 1.6.0 -Publish
```

- `-Publish` — пересобрать приложение (self-contained win-x64). Если уже собрано — можно убрать.
- Готовый файл: `installer\output\CryptoAITerminal-Setup-1.6.0.exe`.

Этот файл и отдаёшь заказчику. Он запускает → «Далее» → готово, ярлык на рабочем столе.
Установка **без прав администратора** (ставится в профиль пользователя), UAC не спрашивает.

> Без подписи Windows SmartScreen всё ещё может ругнуться один раз («Подробнее → Всё
> равно выполнить»). Но это один файл, не россыпь, и репутация набирается быстрее.

## Подпись (убирает «обезвредили» по-настоящему)

### 1. Купить сертификат code-signing
- **OV** (~$200–400/год, Sectigo/DigiCert/…): дешевле, но репутация SmartScreen
  набирается со временем (по числу скачиваний).
- **EV** (~$300–600/год, нужен USB-токен): **сразу** без предупреждений SmartScreen.
  Для раздачи многим заказчикам — лучший вариант.

Где брать: Sectigo, DigiCert, SSL.com, GlobalSign (или реселлеры вроде Cheapsslsecurity).

### 2. Подписать при сборке
Положи сертификат и задай переменные окружения, затем собери — скрипт подпишет и
само приложение, и установщик автоматически:

```powershell
$env:CODESIGN_PFX  = "C:\path\to\cert.pfx"
$env:CODESIGN_PASS = "пароль-от-pfx"
pwsh installer\build-installer.ps1 -Version 1.6.0 -Publish
```

Или по отпечатку сертификата из хранилища (EV-токен обычно так):

```powershell
$env:CODESIGN_THUMBPRINT = "AB12CD...."   # отпечаток сертификата
pwsh installer\build-installer.ps1 -Version 1.6.0 -Publish
```

Проверить подпись:
```powershell
& "C:\Program Files (x86)\Windows Kits\10\bin\10.0.26100.0\x64\signtool.exe" verify /pa installer\output\CryptoAITerminal-Setup-1.6.0.exe
```

## Пока сертификата нет — что сказать заказчику
1. Распаковать/запустить установщик (не из архива — сначала «Извлечь всё», если zip).
2. SmartScreen → «Подробнее» → «Всё равно выполнить».
3. Если антивирус удаляет файлы — добавить папку установки в исключения и переустановить.
4. Поставить Microsoft Visual C++ Runtime x64: https://aka.ms/vs/17/release/vc_redist.x64.exe

## Технические детали
- Скрипт: `build-installer.ps1` — публикует → (подписывает exe) → собирает Inno → (подписывает установщик).
- Скрипт Inno: `CryptoAITerminal.iss` — `AppId` фиксированный (не менять, иначе апдейты
  будут ставиться рядом, а не заменять). Версия и папка-источник передаются через `/D`.
- Иконка: `CryptoAITerminal.TerminalUI\Assets\app.ico`.
- Выход сборки: `installer\output\` (в `.gitignore`).
